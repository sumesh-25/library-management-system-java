# library-managment-system-java
Console-based Library Management System built using Java OOP concepts.

## Features
- Add Student
- Add Teacher
- Search User
- Display Users
- Add Book
- Remove Book
- Search Book
- Display Books
- Issue Book
- Return Book
- Display Issued Books
- Display Available Books
- Books Issued to User

## Concepts Used
- Classes and Objects
- Encapsulation
- ArrayList
- OOP Principles

## Future Improvements
